A very simple parser for a theoretical format called as-is for my CIT 384 class
* Example Input
#+BEGIN_SRC http
status: 200 Cool
X-hostname: cit384-${USER}.csun.edu
content-type: text/plain

A plain text file
#+END_SRC
* Example Output
#+BEGIN_SRC http
HTTP/1.1 200 Cool
server: premortal/cit-384/${USER}
date: ${DATE}
X-hostname: cit384-${USER}.csun.edu
content-type: text/plain

A plain text file
#+END_SRC
